What is 2-D DP?

Tackle grid and matrix problems with two-dimensional DP.

Trigger Words

When you see these in a problem, think 2-D DP:

Template Code

def uniquePaths(m: int, n: int) -> int:
    # Space-optimized 1D DP
    dp = [1] * n
    
    for i in range(1, m):
        for j in range(1, n):
            dp[j] += dp[j - 1]
    
    return dp[n - 1]
